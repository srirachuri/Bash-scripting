# 📅 Day 3 — Viewing & Reading Logs (Bash)

## 📌 Overview
On Day 3, I practiced reading and analyzing log files using Linux commands.  
This is important for monitoring systems and detecting issues in real-world environments.

---

## 🧠 Concepts Learned

- Viewing files using `cat`
- Reading large files using `less`
- Viewing first lines using `head`
- Viewing last lines using `tail`
- Monitoring logs in real-time using `tail -f`

---

## 💻 Commands Practiced

```bash
cat file.log
less file.log
head file.log
head -n 2 file.log
tail file.log
tail -n 2 file.log
tail -f file.log
